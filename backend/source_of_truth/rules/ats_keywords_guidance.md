# ATS Keywords Guidance

When relevant and truthful, prioritize terms aligned to L1 engineering roles:

- first-level technical support
- incident management
- escalation (L2/L3)
- SOP / runbook adherence
- ticketing systems
- troubleshooting
- documentation / knowledge base
- customer-first service
- collaboration and communication
- Windows / Linux / macOS
- networking fundamentals (TCP/IP, DNS)
- monitoring and system reliability

Use concise, natural phrasing and avoid keyword stuffing.
