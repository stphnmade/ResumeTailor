# Tailoring Rules

- Use only verified information from the input resume and canonical source resume.
- Optimize for ATS alignment by prioritizing role-relevant keywords from the job description.
- Preserve LaTeX structure and readability.
- Prefer tailoring within the canonical section structure and macro layout instead of inventing a new visual design.
- Keep output to one page; if content risks spilling onto a second page, trim lower-relevance content aggressively.
- Favor a smaller set of highly relevant experiences and projects over broad completeness.
- Prefer 2 sharply aligned projects unless a third clearly fits within one-page constraints.
- Rewrite and compress bullets before expanding them; dense, credible phrasing is better than exhaustive detail.
- Prioritize L1 support signals: first-level support, incident handling, escalation quality, SOP adherence, documentation discipline, troubleshooting, and customer satisfaction.
- Keep metrics exactly as provided; do not alter numeric values.
- User recruiter-instruction notes may guide emphasis, ordering, and phrasing, but must never override truth or one-page constraints.
